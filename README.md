# Playwright Automation Demo 🚀

This repository showcases a professional automation framework using **Playwright** and **JavaScript**. 

## 🏗️ Architecture: Page Object Model (POM)
I implemented the **Page Object Model** design pattern to ensure:
* **Maintainability:** Clean separation between test logic and locators.
* **Reusability:** Page classes can be shared across multiple tests.
* **Readability:** Tests are easy to understand for any stakeholder.

## 🛠️ Tools & Technologies
* **Language:** JavaScript (Node.js)
* **Testing Framework:** Playwright
* **Pattern:** Page Object Model (POM)
